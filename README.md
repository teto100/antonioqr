# QR Tech Test System

Sistema de evaluación técnica desarrollado con Next.js 14, TypeScript y Firebase.

## Características

- ✅ **Next.js 14** con TypeScript
- ✅ **Firebase** para base de datos
- ✅ **Tailwind CSS** para estilos
- ✅ **Sistema de seguridad** integrado
- ✅ **Responsive design**
- ✅ **Rate limiting** anti-DoS

## Instalación

```bash
npm install
npm run dev
```

## Estructura del Proyecto

```
src/
├── app/
│   ├── api/          # API Routes
│   └── (pages)/      # Páginas
├── components/       # Componentes React
├── hooks/           # Custom hooks
└── lib/             # Utilidades
```

## Tecnologías

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Backend**: Firebase Firestore
- **Deployment**: Vercel
- **Seguridad**: Rate limiting, validaciones, tokens seguros
